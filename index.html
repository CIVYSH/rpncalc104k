Info

RPNCalc104k is a calculator for Android which uses the Reverse Polish Notation to perform the calculations. That is, if you want to get the surface of a circle with radius 104 you will do:

S = PI * r²

or even easier:

This piece of software is licensed under the GPLv3. That means you can download and install it for free and enjoy its features:

  • Basic operations ( + * sqrt ... )
  • Trigonometric functions ( sin cos tan rad/deg ... )
  • Extra functions ( log mean surface-of-a-triangle ... )
  • History saver to go back and fordward
